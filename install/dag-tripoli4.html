Install DAG-Tripoli4¶

This document explains how to install DAG-Tripoli4, assuming you have already installed the required dependencies.

DAG-Tripoli4 uses Tripoli4, which is is distributed by CEA/Saclay as a binary executable. For access to Tripoli4, please contact Jean-Christophe Trama.
